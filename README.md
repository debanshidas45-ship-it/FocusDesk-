Developed a web-based timer application that allows users to set custom countdown durations with an intuitive and interactive interface. The application features editable time inputs, real-time countdown updates, and control options such as pause and reset. Upon completion of the timer, an alert sound and background music are triggered, with a user-controlled option to stop the audio. The interface is styled with a modern glassmorphism design and a nature-themed background, enhancing the overall user experience.
